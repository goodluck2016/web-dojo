# web-dojo

A dojo website for JavaScript and more.

Used Bootstrap 3、Vue2、Ace Editor and more.


# Two Parts

## [/server](server)

server-side for web-dojo, provider some rest api.

## [/client](client)

client-side for web-dojo.

## Browser support

- Chrome latest
- Firefox latest
- Microsoft EDGE
- IE 10+ (Include IE10)
